About

This website was created on the 22nd June 2022 and is being updated constantly to house newer and more exciting projects!

The Birth of This website

I created this website after a STEM Venturi club where we used Anvil.works to create a website using Python. However, although the designing of the site was very intuitive, I had no basic knowledge of HTML and I was unable to fully exploit the capabilities of Anvil.

Therefore I chose to take the arduous yet incredibly satisfying journey of building my own website! The goal for this website is to be able to house most of my code so it can be run remotely, especially maths functions (as the website header suggests). Also, in the future I may use this website as my portfolio to showcase a variety of different things, making it easier for me to represent myself.
